message 消息提示
组件说明
常用于主动操作后的反馈提示。与 Notification 的区别是后者更多用于系统级通知的被动提醒。
基础用法
从顶部出现，3 秒后自动消失。
<button class="ape-button ape-button-plain-default" onclick="open1()">打开消息提示</button>         
function open1() {
    $message("这是一条消息提示。")
}
不同状态
用来显示「成功、警告、错误、信息」类的操作反馈。
<button class="ape-button ape-button-success" onclick="open2('success','成功')">成功</button>
<button class="ape-button ape-button-warning" onclick="open2('warning','警告')">警告</button>
<button class="ape-button ape-button-error" onclick="open2('error','错误')">错误</button>
<button class="ape-button ape-button-info" onclick="open2('info','信息')">信息</button>                   
function open2(type, message) {
    $message({
        title: `这是一条 ${message} 消息`,
        type: type
    })
}
不同时长
动态控制message展示的时长。
<button class="ape-button  ape-button-success" onclick="open3('success','成功',3000)">成功 3s</button>
<button class="ape-button  ape-button-warning" onclick="open3('warning','警告',2000)">警告2s</button>
<button class="ape-button  ape-button-error" onclick="open3('error','错误',1000)">错误 1s</button>            
function open3(type, message, duration) {
    $message({
        title: `这是一条 ${message} 消息`,
        type: type,
        duration: duration
    })
}
属性
参数 说明 类型 可选值 默认值
title 消息文字 string —— ——
type 类型 string success/warning/info/error success
duration 显示时间, 毫秒。 number —— 3000
