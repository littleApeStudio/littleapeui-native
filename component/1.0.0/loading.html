loading 加载
组件说明
加载数据时显示动效。
基础用法
在网络请求或者其他操作时为避免用户未知情况下的等待，可增加交互体验。
<button class="ape-button ape-button-plain" onclick="open1()">打开加载</button>
function open1() {
    $loading.show()
    setTimeout(() => {
        $loading.hide()
    }, 5000);
}
自定义用法
可自定义设置加载提示文本。
<button class="ape-button ape-button-plain" onclick="open2('我是自定义的提示文本')">打开自定义文本加载</button>
function open2(e) {
    $loading.show(e)
    setTimeout(() => {
        $loading.hide()
    }, 5000);
}
