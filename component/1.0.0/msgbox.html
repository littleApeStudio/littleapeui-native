msgbox 弹框
组件说明
模拟系统的消息提示框而实现的一套模态对话框组件，用于消息提示、确认消息和提交内容。
消息提示
—当用户进行操作时会被触发，该对话框中断用户操作，直到用户确认知晓后才可关闭。
<button class="ape-button ape-button-plain-default" onclick="open1()">打开 msgbox 弹窗</button>           
function open1() {
    $msgbox({
        title: "标题名称",
        content: "这是一段内容"
    }).then(() => {
        $message("点击了确定")
    }).catch(() => {
        $message("点击了取消", "warning")
    })
}
不同状态
用来显示「成功、警告、错误」类的对话框。
<button class="ape-button ape-button-success" onclick="open2('success','成功')">成功</button>
<button class="ape-button ape-button-warning" onclick="open2('warning','警告')">警告</button>
<button class="ape-button ape-button-error" onclick="open2('error','错误')">错误</button>           
function open2(type, message) {
    $msgbox({
        title: `${message}`,
        content: `这是 ${message} 的对话框`,
        type: type
    })
}
自定义按钮文字
可以分别设置取消按钮和确定按钮的文字。
<button class="ape-button" onclick="open3()">更改按钮名字</button>           
function open3() {
    $msgbox({
        confirmText: "改了确定",
        cansoleText: "改了取消",
    })
}
显示/隐藏取消按钮
动态控制取消按钮的显隐。
<button class="ape-button" onclick="open4(true)">显示</button>
<button class="ape-button" onclick="open4(false)">隐藏</button>           
function open4(e) {
    $msgbox({
        showCansole: e
    })
}
显示/隐藏关闭按钮
动态控制关闭按钮的显隐。
<button class="ape-button" onclick="open5(true)">显示</button>
<button class="ape-button" onclick="open5(false)">隐藏</button>           
function open5(e) {
    $msgbox({
        showClose: e
    })
}
属性
参数 说明 类型 可选值 默认值
title 弹窗标题 string —— ——
type 类型 string success/warning/error success
content 弹窗内容 string —— ——
confirmText 确定按钮文字 string —— 确定
cansoleText 取消按钮文字 string —— 取消
showCansole 是否显示取消按钮 boolean —— true
showClose 是否显示关闭按钮 boolean —— true
