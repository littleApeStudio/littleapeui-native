快速上手
本节将介绍如何在项目中使用 littleApe-UI。
示例--普通 dom 元素
你可以引入整个 littleApe-UI 组件。然后只需要给对应的 dom 元素赋予对应的 class 类名即可，该示例将展示如何引用 littleApeUI 的警告按钮。
<body>
    <button class="ape-button ape-button-warning">警告按钮</button>
</body>
示例--js 交互函数
想要使用 littleApeUI 的组件只需要调用对应的函数即可。
<body>
    <button class="ape-button ape-button-plain-default" onclick="open1()">打开 msgbox 弹窗</button>
</body>


<script>
    function open1() {
        $msgbox({
            title: "标题名称",
            content: "这是一段内容"
        }).then(() => {
            $message("点击了确定")
        }).catch(() => {
            $message("点击了取消", "warning")
        })
    }
</script>
